# WhatsApp Opt-In Broadcast Bot

A WhatsApp automation bot designed for **opt-in broadcasts, notifications, and media delivery** with built-in safety mechanisms.

## Features
- Opt-in / opt-out keywords (START / STOP)
- Broadcast messages to subscribed users only
- Message delays & throttling
- Media support (text, image, video)
- Safe daily limits
- Node.js based WhatsApp integration

## Intended Use
This project is designed for:
- Product updates
- Service notifications
- User alerts
- Opted-in broadcasts

❌ Not intended for spam or cold messaging.

## Safety
- Users must initiate conversation
- Every message supports unsubscribe
- Rate limits must be respected
